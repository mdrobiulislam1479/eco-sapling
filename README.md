Eco Sapling is a clean, modern plant shop experience built with Next.js. It
highlights curated greenery, care guidance, and a smooth browsing flow across
home, collections, and plant detail views.

## Live URL: https://eco-sapling.vercel.app

## Setup and installation

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Run the production server:

```bash
npm run start
```

## Routes

- `/` Home page with marketing sections and featured plants.
- `/plants` Plant catalog with popular listings.
- `/plants/[id]` Plant detail page for individual items.

## Implemented features

- Hero banner with featured imagery and CTA.
- Popular plants grid powered by local JSON data.
- Eco promise, styling, care rituals, and community storytelling sections.
- Newsletter call-to-action block.
- Shared navigation and footer across pages.
- App-level error and not-found pages.

## Feature notes

- Home sections reinforce the brand story and guide users through discovery.
- The plants grid reuses a card component for consistent visuals.
- Error and 404 pages offer clear recovery actions for better UX.
